Books

All these books about Python and R can be accessed for free. They are open access and open source.

This is collection of works that I consult almost daily while working.

R

  • Introduction to Data Science - A detailed introduction to Data science by a biostatistician.
  • Advanced R - All you wish to know about programming in R.
  • Introduction to Statistical Learning - A detailed introduction to modern statistical methods, implemented in R.
  • Text Mining in R Analyzing natural language and written text in R.
  • Tidy Modeling with R R’s machine learning framework.
